# Landing Page de Serviços – Saúde Pro

## 1. Descrição do Projeto
Este projeto consiste no desenvolvimento de uma **Landing Page profissional de Serviços de Saúde**, criada para fins acadêmicos na disciplina de **DevOps**.  
O objetivo é aplicar boas práticas de desenvolvimento front-end, organização de repositório Git e preparar a estrutura para futuras implementações de **CI/CD, containers e monitoramento**.

O site apresenta informações institucionais, serviços oferecidos, seção de perguntas frequentes, formulário de contato validado e recursos de interatividade com JavaScript.

---

## 2. Modelo de Negócio
**Nicho escolhido:** Saúde  

A empresa fictícia atua na prestação de serviços de saúde com foco em atendimento humanizado, prevenção e orientação ao paciente.

---
